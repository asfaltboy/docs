If your application is deployed via a Function-as-a-Service (FaaS) offering and uses a standard Node.js runtime, it is a serverless app. Common deployment examples include [AWS Lambda](./deploying-to-aws-lambda) and [Vercel Serverless Functions](./deploying-to-vercel).
