<TopBlock>

If your application is deployed via an "Edge Function" offering or is deployed from a [serverless](../serverless) offering and has a non-standard runtime, it is a edge deployed app. Common examples include [Cloudflare Workers](./deploying-to-cloudflare-workers) and Vercel Edge Functions.

</TopBlock>
